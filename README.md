# Web LAB-1 - Hello Project

## Hakkinda
Bu proje, Web Tasarimi ve Programlama dersi LAB-1 / LAB-2 / LAB-3 / LAB-4 kapsaminda
Vite + React + TypeScript + Tailwind CSS kullanilarak olusturulmustur.

## Kullanilan Teknolojiler
- React 19
- TypeScript
- Vite
- Tailwind CSS v4

## Kurulum
```bash
npm install
```

## Calistirma
```bash
npm run dev
```
Tarayicida http://localhost:5173 adresini ac.

## Ozellikler
- Semantik HTML5 yapisi (header, nav, main, section, article, footer)
- Erisilebirlik (a11y): skip link, aria-label, heading hiyerarsisi, form label iliskisi
- Responsive tasarim: 3 breakpoint (mobil / tablet / masaustu)
- Tailwind CSS v4 ile utility-first stillendirme
- Dark mode destegi (tema degistirme butonu)
- Component kutuphanesi: Button, Input, Card, Alert
- UI Kit sayfasi: 8+ bilesen varyanti
