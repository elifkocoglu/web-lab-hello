# Web Tasarımı ve Programlama LAB-1

## Proje Hakkında
Bu proje, React ve TypeScript kullanılarak oluşturulmuş temel bir web uygulamasıdır. 
Vite araç kiti kullanılarak geliştirme ortamı yapılandırılmıştır.

## Projeyi Çalıştırma
Projeyi yerel ortamınızda çalıştırmak için:
1. `npm install` (Bağımlılıkları yükler)
2. `npm run dev` (Sunucuyu başlatır)
